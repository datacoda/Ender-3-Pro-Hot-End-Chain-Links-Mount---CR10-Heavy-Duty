# Summary

Remixed FedorSosnin's [strain relief mount v8](https://www.thingiverse.com/thing:3132855) for use with the [CR-10 Heavy Duty e3D V6 mount](https://www.thingiverse.com/thing:2494642) from chito.  It takes a notch out of the original straddle to avoid conflict with the V6 base mount.

I also removed the hex cutouts (unneeded complexity) as well as beefed up the riser.

This has only been tested on a Ender 3 Pro.  This thing only has the modified hot-end-side-link.  See sources for other parts.
